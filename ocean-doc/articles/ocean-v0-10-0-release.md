# What's New in Ocean v0.10.0

A major milestone release introducing workspace management and advanced merge tools.

## Key Highlights

- **Multi-Workspace Dashboard**: Manage dozens of repositories simultaneously from a unified dashboard. Switch between projects instantly with `Cmd+K`.
- **AI-Assisted 3-Way Merge**: A revolutionary new visualizer for resolving merge conflicts. Ocean uses AI to understand the intent behind conflicting changes and suggests the most logical resolution.

## Features

- New "Compact Mode" for the sidebar.
- Integrated performance monitoring for long-running agent tasks.
- Enhanced security for environment variable storage.

Happy coding!
