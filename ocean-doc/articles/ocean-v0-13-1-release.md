# Ocean v0.13.1 — What's New

Released: 2026-04-25

## Parallel Claudes

The headline feature of v0.13.1 is **Parallel Claudes**. This allows developers to orchestrate multiple Claude agents simultaneously to solve complex engineering problems.

- **Batch Spawn** — Create multiple Claude sessions with a single command or UI interaction.
- **Broadcast Commands** — Send the same prompt or instruction to all active Claude sessions at once.
- **N-Way Diff** — Compare the code generated by different agents side-by-side to identify the best implementation.
- **Promote Winner** — With one click, merge the winning agent's work into the base while discarding the others.
