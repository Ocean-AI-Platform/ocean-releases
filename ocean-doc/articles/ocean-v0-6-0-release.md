# What's New in Ocean v0.6.0

Our biggest bug-fix release yet, with a focus on visual feedback.

## Key Highlights

- **Git Visualizer**: A beautiful, interactive graph of your Git history directly in the terminal sidebar.
- **Split Indicators**: Visual markers in the gutter showing which parts of a file have been modified by an agent versus a human.
- **16 Critical Bug Fixes**: Including terminal resizing improvements and fix for the "Ghost Terminal" PTY issue.

## Performance

- Reduced CPU overhead for background file watching.
- Faster workspace loading for repositories with >100k files.

Enjoy the new Ocean!
