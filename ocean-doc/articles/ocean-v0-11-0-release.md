# What's New in Ocean v0.11.0

We are excited to announce the release of Ocean v0.11.0, featuring significant improvements to the developer experience and infrastructure management.

## Key Highlights

- **MCP Server Config Editor**: A new integrated UI for managing your Model Context Protocol (MCP) server configurations. Easily add, edit, and toggle servers without touching JSON files manually.
- **Persistent Session Logs**: Your session history and AI agent logs are now persisted across application restarts. Never lose context of a complex debugging session again.
- **Port-Forwarding Dashboard**: A dedicated view to manage active port forwards. See all exposed ports in one place and manage their lifecycle with a single click.

## Bug Fixes and Improvements

- Optimized log rotation to prevent excessive disk usage.
- Improved the startup speed of the internal PTY engine.
- Fixed a rendering glitch in the workspace switcher.

Thank you for being part of the Ocean community!
