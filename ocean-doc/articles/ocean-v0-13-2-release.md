# Ocean v0.13.2 — What's New

Released: 2026-04-27

## Release Overview

Ocean v0.13.2 focuses on infrastructure hardening and documentation for the next phase of the project. This release ensures that all builds undergo comprehensive key verification and restores development environment identifiers.

## Key Changes

- **Comprehensive Key Verification** — Added mandatory key verification steps before the build process to ensure supply chain security for all signed builds.
- **Project Documentation** — Added a comprehensive fundraising package including a pre-seed investor memo, one-pager, and private deal notes to the project repository.
- **Environment Restoration** — Restored development identifiers and icons following the production release cycle.

## Infrastructure Improvements

- Hardened the release automation pipeline with additional pre-flight checks.
- Synchronized environment secrets across the WorkOS split.
