# What's New in Ocean v0.7.1

Expanding the Ocean ecosystem with better UI interactions and testing infrastructure.

## Key Highlights

- **Context Menus**: Right-click support added throughout the terminal and sidebar for quick access to Git commands and file operations.
- **Merge & Ship Workflow**: Enhanced the automated workflow for merging and deploying code directly from the terminal.
- **E2E Test Suite**: Significant expansion of our internal End-to-End test suite to ensure long-term stability of the PTY engine.

## Infrastructure

- Rust backend performance optimizations.
- Improved memory management for large file diffs.

Thanks for your support!
