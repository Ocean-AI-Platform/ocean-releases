# What's New in Ocean v0.7.0

A massive update focusing on conflict resolution and developer visualization.

## Key Highlights

- **Advanced Conflict Resolution**: Integrated support for resolving complex Git conflicts with AI-powered suggestions.
- **AI Merge**: One-click merging of feature branches with intelligent conflict pre-screening.
- **File Locks**: Prevent agents from overwriting sensitive files during high-concurrency tasks.
- **Heatmap & Dependency Graph**: Visualize which files are being touched most frequently and how they relate to the rest of your codebase.

## Additional Features

- Integrated port forwarding.
- Expanded documentation and keyboard shortcut guides.

Ocean is getting better every day!
