# What's New in Ocean v0.10.1

This patch release focuses on stability and integration improvements.

## Key Highlights

- **Git Index Lock Fixes**: Resolved a critical issue where parallel commits from multiple agents could leave the Git index in a locked state.
- **Cursor CLI Detection**: Ocean now automatically detects if you have the Cursor editor installed and allows you to open files directly in Cursor from the terminal.

## Minor Updates

- Updated the default prompt templates for better performance with Claude 3.5 Sonnet.
- Fixed a rare crash when closing a workspace with active background processes.

Stay tuned for more updates!
